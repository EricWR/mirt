MDIFF {mirt}R Documentation

Compute multidimensional difficulty index

Description

Returns a matrix containing the MDIFF values (Reckase, 2009).

Usage

MDIFF(x)

Arguments

x

an object of class 'SingleGroupClass'

Author(s)

Phil Chalmers rphilip.chalmers@gmail.com

References

Reckase, M. D. (2009). Multidimensional Item Response Theory. Springer.

See Also

extract.group, MDISC

Examples

## No test: 

mod <- mirt(Science, 2)
MDIFF(mod)

##            MDIFF_1    MDIFF_2   MDIFF_3
## Comfort -2.9082859 -1.5993792 0.8946023
## Work    -0.8809265 -0.2742757 0.6907591
## Future  -1.3261155 -0.5563912 0.4935955
## Benefit -1.3450947 -0.4030597 0.6892745

## End(No test)
